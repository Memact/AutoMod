# Branding And Asset Use

The code in this repository is open source under [Apache-2.0](LICENSE).

Memact branding is not open source. Unless a specific file says otherwise, the
following are excluded from the open-source license and are all rights reserved:

- the `Memact` name and related product names
- logos, icons, wordmarks, banners, and other visual identity elements
- screenshots, promotional images, and other Memact-owned artwork

You may:

- fork and modify the code
- deploy your own copy of the bot
- mention Memact only to truthfully describe the source of the project

You may not:

- reuse Memact branding or assets as your own
- imply your fork is the official Memact project
- redistribute Memact-owned brand assets without permission

If you publish a fork, remove or replace Memact branding and assets unless you
have written permission to keep them.
