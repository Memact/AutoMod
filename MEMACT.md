# Memact org context

**Permissioned intent infrastructure for apps.**

```text
Understand what users are trying to do.
```

Memact is infrastructure that helps apps predict user intent from approved digital activity, without giving them raw access to a user's private data.

This repo is not part of the core Memact intent infrastructure. It is a separate Discord moderation and community-operations project under the same GitHub organization.

## Core Memact system

```text
Access -> Capture -> Inference -> Schema -> Memory -> Intent
```

AutoMod should not be described as a Memact memory, capture, inference, schema, access, or intent layer.
